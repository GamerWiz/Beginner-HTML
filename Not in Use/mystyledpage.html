How do you use CSS?

Selectors

"Selectors" are basically Java if statements

      selector {
        properties
      }

For example, if I wanted all body text to be blue, I would write

      body{
       color: blue;
      }

Units

  • px, em (multiply font size), pt, cm, mm, percentages are all units
  • No need for units if using 0
